# Data Directory

This folder groups the datasets used across the project.

- `DataSetTeensyv9_ULTRA_CLEAN.csv` is the default dataset used by the current `src/` pipeline.
- The other `DataSetTeensy*.csv` and `dataset_correctif_*.csv` files are historical iterations kept for comparison and traceability.

If you only need one dataset to start from, use `DataSetTeensyv9_ULTRA_CLEAN.csv`.
